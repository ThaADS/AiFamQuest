## Checklist Verification Tasks

### Before Launch
1. [ ] Run Lighthouse audit (all pages)
   - Open Chrome DevTools
   - Go to Lighthouse tab
   - Run audit
   - Target: All scores > 90

2. [ ] Test mobile responsiveness
   - Chrome DevTools (F12)
   - Toggle device toolbar
   - Test all pages on iPhone, Android sizes

3. [ ] Verify all links
   - Internal links work
   - External links open in new tab
   - No 404 errors

4. [ ] Check OpenGraph preview
   - Use og:image debugger
   - Facebook Sharing Debugger
   - Twitter Card validator

5. [ ] Test Core Web Vitals
   - PageSpeed Insights
   - Web Vitals Chrome Extension
   - Verify LCP < 2.5s

6. [ ] Accessibility audit
   - WAVE browser extension
   - Axe DevTools
   - Manual keyboard navigation

7. [ ] Search Console setup
   - Verify site ownership
   - Submit sitemap
   - Check indexing status

8. [ ] Structured data validation
   - Schema.org validator
   - Check JSON-LD markup
   - Test Rich Result tests

9. [ ] Security check
   - SSL certificate valid
   - No mixed content warnings
   - Check security headers

10. [ ] Content review
    - Spelling/grammar check
    - Meta descriptions 155 chars max
    - H1 unique per page
    - Image alt text complete

### Post-Launch (Monthly)
- [ ] Monitor Search Console
