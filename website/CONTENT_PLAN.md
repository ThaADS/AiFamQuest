## Blog Post Structure

### Template

```markdown
# [H1 - Primary Keyword]

## Introduction
- Hook (relatable problem or statistic)
- Pain point
- Promise of solution
- Length: 100-150 words

## [H2 Main Section 1]
- 200-300 words
- Practical advice
- Examples
- Subheadings (H3) as needed

## [H2 Main Section 2]
- Similar format

## [H2 Main Section 3]
- Similar format

## Key Takeaways
- 3-5 bullet points
- Actionable
- Memorable

## Call-to-Action
- Soft CTA (non-pushy)
- Relevant to article
- Internal link

## Related Articles
- 2-3 links to related posts

## Resources
- External links (quality sources)
- Tools mentioned
- Further reading
```

### SEO Checklist per Article

- [ ] Keyword in H1
- [ ] Keyword in meta title (50-60 chars)
- [ ] Keyword in meta description (155-160 chars)
- [ ] Keyword in first 100 words
- [ ] Internal links (3-5)
- [ ] External links (2-3)
- [ ] Images with alt text
- [ ] 1000+ words
- [ ] Readable (short paragraphs, bullets)
- [ ] Unique angle vs competitors
- [ ] Data/statistics with sources
- [ ] Author info with bio
- [ ] Publication date
- [ ] Updated date (if revised)
